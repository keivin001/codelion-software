Healthcare Sector

HealthTrack Rwanda

An enterprise patient metrics dashboard and localized syncing system designed to streamline check-in pipelines across decentralized clinics.

The Challenge

Rural health clinics in Rwanda struggled with intermittent internet connections, resulting in duplicate patient files, mismatched health metrics, and long queues at hospital registration desks.

Our Solution

We developed an offline-first syncing database architecture that runs local SQLite databases on clinics' computers. When internet is active, data synchronizes to a main PostgreSQL database using conflict-free replicated data types (CRDTs). We also built a clean web dashboard for health workers.

The Outcome

60% Reduction in Queue Wait Times | 100% Data Integrity

CategoryTechnologies Utilized
BackendNode.js, Express, Socket.io
DatabasesPostgreSQL, SQLite, Redis Cache
FrontendReact.js, TailwindCSS, Chart.js
Fintech & Payments

PayLink Africa

A unified payments portal coordinating seamless checkouts and payouts between global merchants and mobile networks.

The Challenge

E-commerce platforms and global businesses wanted to bill customers in East Africa but had difficulties integrating with dozens of mobile money (MTN Mobile Money, Airtel Money) and local banking APIs individually.

Our Solution

We built a scalable, highly secure integration gateway that wraps local telecom payment systems and banks in a developer-friendly REST API. We implemented double-entry ledger designs and automated webhooks to ensure payment reconciliation with sub-millisecond latencies.

The Outcome

$2M+ USD Transaction Volume | 99.99% Integration Uptime

CategoryTechnologies Utilized
BackendGo (Golang), GORM
HostingAWS Elastic Kubernetes Service (EKS)
SecurityOAuth2, AES-256 Encryption, Webhooks
EdTech & Education

EduBridge LMS

A learning management system hosting interactive coursework, progress visualizations, and certificate issuing structures.

The Challenge

An educational institute needed to host content for 10k+ active learners. However, large PDF downloads and heavy video loading times crashed standard VPS servers during exam seasons.

Our Solution

We decoupled the learning app architecture. Course assets are now cached on globally distributed CDN nodes. We utilized Node.js clustering and database indexing to optimize server operations. We also created a custom mobile application with local offline storage.

The Outcome

10k+ Active Students Supported | 80% Reduction in Server Costs

CategoryTechnologies Utilized
FrameworkNext.js, React, Node.js
InfrastructureCloudflare CDN, Google Cloud Platform
DatabaseMongoDB Atlas, Redis
